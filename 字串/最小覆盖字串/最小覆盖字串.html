🔍 最小覆盖子串 · 滑动窗口算法

来自 LeetCode 76 题 | Python 实现 (collections.defaultdict + 双指针) | 交互可视化逐步执行

from collections import defaultdict
class Solution:
    def minWindow(self, s: str, t: str) -> str:
        d = defaultdict(int)
        for char in t: d[char] += 1
        formed, total = 0, len(d)
        l = r = 0
        len_ans = float('inf')
        subl, subr = 0, 0
        while r < len(s):
            char = s[r]
            if char in d:
                d[char] -= 1
                if d[char] == 0: formed += 1
            while l <= r and formed == total:
                curr_len = r - l + 1
                if curr_len < len_ans:
                    len_ans, subl, subr = curr_len, l, r+1
                char_l = s[l]
                if char_l in d:
                    if d[char_l] == 0: formed -= 1
                    d[char_l] += 1
                l += 1
            r += 1
        return "" if len_ans == float('inf') else s[subl:subr]

💡 算法核心：
1️⃣ 右指针 r 扩张，加入字符并更新需求字典。
2️⃣ 当所有字符都满足 (formed == total) 时，尝试缩小左指针 l 以寻找更短的可行窗口。
3️⃣ 每次收缩前记录最小窗口，同时恢复字典计数，直到窗口不再覆盖。
4️⃣ 最终返回最小覆盖子串。

📘 变量含义对照
  • d : 每个目标字符的剩余需求量 (负数表示超出需求)
  • formed : 已经完全满足需求的不同字符个数 (d[char]==0)
  • total : t 串中不同字符总数
  • l , r : 滑动窗口左右边界 (包含)
  • len_ans : 当前找到的最短窗口长度
  • subl, subr : 最佳窗口切片索引 (右开区间)
